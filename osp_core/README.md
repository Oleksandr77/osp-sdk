# OSP Core SDK

The official Python SDK for the [Open Skills Protocol](https://github.com/antigravity/osp).

## Features
- **JCS**: JSON Canonicalization Scheme implementation.
- **Crypto**: Utilities for signing and verifying OSP messages.
- **Models**: Pydantic models for OSP primitives (Agents, Skills, Messages).

## Installation
```bash
pip install osp-core
```

## Usage
```python
from osp_core.crypto import JCS
# ...
```
